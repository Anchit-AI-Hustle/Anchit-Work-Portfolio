# Push, deploy & permanent auto-sync (via GitHub CLI)

Everything below runs **on your Mac**, where your GitHub + Vercel logins live.
This Cowork session can't push or deploy directly (no access to those
credentials) — the GitHub CLI (`gh`) on your Mac provides the auth.

- Repo: https://github.com/Anchit-AI-Hustle/Anchit-Work-Portfolio
- Vercel project: `anchit-work-portfolio` (git-connected, so **a push auto-deploys**)
- Files: `scripts/gh-sync.sh` (worker) · `scripts/com.anchit.portfolio.autosync.plist` (schedule)

---

## 1. One-time: authenticate the GitHub CLI

Open **Terminal**:

```bash
# install gh if you don't have it
brew install gh

# log in (pick: GitHub.com → HTTPS → login with browser)
gh auth login

# let git use gh's credentials for push/pull
gh auth setup-git
```

## 2. Push + deploy right now

```bash
cd "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio"
git add -A
git commit -m "Portfolio: Indian-voice narration, chat as floating CTA, UX fixes"
git push origin main
```

The push auto-triggers a **production deploy** on Vercel — no extra step.
(Force a deploy without a push anytime with `npx vercel deploy --prod`.)

## 3. Turn on permanent auto-sync

Installs a `launchd` agent that runs `scripts/gh-sync.sh` at login and every 10
minutes. Any change in this folder — yours or ones I make in a future session —
gets committed, pushed, and (via Vercel) redeployed automatically.

```bash
chmod +x "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio/scripts/gh-sync.sh"

cp "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio/scripts/com.anchit.portfolio.autosync.plist" ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.anchit.portfolio.autosync.plist
```

Done — hands-off from here.

## Handy commands

```bash
# sync right now instead of waiting for the timer
bash "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio/scripts/gh-sync.sh"

# see what the agent did
cat /tmp/anchit-portfolio-autosync.out /tmp/anchit-portfolio-autosync.err

# pause / resume
launchctl unload ~/Library/LaunchAgents/com.anchit.portfolio.autosync.plist
launchctl load   ~/Library/LaunchAgents/com.anchit.portfolio.autosync.plist

# verify auth anytime
gh auth status
```

## Notes

- `www/` is gitignored — it's regenerated by `npm run build`, which Vercel runs on
  each deploy, so only source (`index.html` etc.) is committed.
- The agent only commits when there are real changes and rebases before pushing,
  so it won't clobber commits made elsewhere.
- `scripts/gh-sync.sh` is the active worker. `scripts/autosync.sh` (pure git) and
  `scripts/claude-sync.sh` (Claude CLI) remain as alternatives — point the plist
  at whichever you prefer.
