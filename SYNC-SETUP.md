# Permanent GitHub auto-sync

This makes **every change in this folder land in GitHub automatically** — including
edits made in future Cowork/Claude sessions, since they happen in this same folder.
It runs on your Mac (where your GitHub login lives) via a `launchd` agent that
auto-commits and pushes every 10 minutes (and at login).

Repo: https://github.com/Anchit-AI-Hustle/Anchit-Work-Portfolio
Files: `scripts/autosync.sh` (the worker) and `scripts/com.anchit.portfolio.autosync.plist` (the schedule).

---

## One-time setup (run in Terminal on your Mac)

Open **Terminal** and run these blocks in order.

### 1. Cache your GitHub credential (so pushes need no password)

```bash
# macOS git ships with the keychain helper — turn it on:
git config --global credential.helper osxkeychain
```

Create a **fine-grained Personal Access Token** at
https://github.com/settings/tokens?type=beta — give it access to the
`Anchit-Work-Portfolio` repo with **Contents: Read and write**.

Then trigger one push so the token gets saved into your Keychain:

```bash
cd "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio"
git add -A
git commit -m "Enable auto-sync + latest portfolio fixes"
git push origin main
```

When prompted: **Username** = your GitHub username, **Password** = paste the token.
(You'll only be asked once — it's cached after that.)

> Prefer the GitHub CLI instead? `brew install gh && gh auth login && gh auth setup-git`
> also works and skips the token-paste.

### 2. Make the script executable

```bash
chmod +x "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio/scripts/autosync.sh"
```

### 3. Install & start the auto-sync agent

```bash
cp "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio/scripts/com.anchit.portfolio.autosync.plist" ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.anchit.portfolio.autosync.plist
```

That's it. From now on, changes auto-commit and push every 10 minutes and at login.

## Useful commands

```bash
# Push right now instead of waiting for the timer:
bash "/Users/anchittandon/Library/Mobile Documents/com~apple~CloudDocs/ANCHIT'S AI HUSTLE/Anchit-Work-Portfolio/scripts/autosync.sh"

# See recent auto-sync output / errors:
cat /tmp/anchit-portfolio-autosync.out /tmp/anchit-portfolio-autosync.err

# Pause auto-sync:
launchctl unload ~/Library/LaunchAgents/com.anchit.portfolio.autosync.plist

# Resume auto-sync:
launchctl load ~/Library/LaunchAgents/com.anchit.portfolio.autosync.plist

# Change the interval: edit StartInterval (seconds) in the plist, then unload + load again.
```

## Notes

- `www/` is gitignored — it's regenerated by `npm run build`, which Vercel runs on
  each push, so only source files (like `index.html`) are committed. Vercel then
  auto-deploys the new build.
- The agent only commits when there are actual changes, and it `pull --rebase`es
  before pushing, so it won't clobber commits you make elsewhere.
- Because the folder is in iCloud Drive, give iCloud a moment to finish syncing
  large edits before expecting them in a commit; the 10-minute cadence absorbs this.
